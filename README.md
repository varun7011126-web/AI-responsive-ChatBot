# AI-responsive-ChatBot

An interactive AI-powered chatbot built using HTML, CSS, and JavaScript, integrated with an  API to generate intelligent, human-like responses.
The project demonstrates real-time user interaction, API integration, and clean UI design.
Google gemini API is used in the chatbot.



Features
Conversational chatbot interface
AI-generated responses using an external API
Real-time message handling
Clean and responsive UI
User-friendly chat layout
Lightweight and fast
